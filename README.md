# Gravity Flip — Neon Mix

**Gravity Flip — Neon Mix** to dynamiczna, neonowa gra typu "endless runner" w przestworzach, w której gracz steruje postacią przeskakującą między podłogą a sufitem, unikając kolców i zdobywając punkty. Wersja jest wzbogacona o efektowne neonowe fale przy flipie oraz ambientową ścieżkę dźwiękową.

---

## Funkcje

- Klasyczna mechanika "gravity flip" — zmiana grawitacji między podłogą a sufitem  
- Losowo generowane przeszkody w postaci kolców  
- Neonowe efekty wizualne przy flipie  
- Ambientowa muzyka i dźwięki flipów  
- Automatyczne zapisywanie najlepszego wyniku w `localStorage`  
- Responsywne UI i skalowanie na różnych urządzeniach  
- Przyjazny dla PWA — możliwość instalacji na telefon
